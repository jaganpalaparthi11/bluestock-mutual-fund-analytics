# Mutual Fund Analytics Platform

## Overview
This project analyzes mutual fund performance, investor behavior, and risk metrics using Python, SQLite, and Power BI.

## Technologies Used
- Python
- Pandas
- NumPy
- SQLite
- Power BI

## Project Structure

data/
notebooks/
dashboard/
reports/
sql/

## Features
- ETL Pipeline
- Data Cleaning
- Exploratory Data Analysis
- Fund Performance Analysis
- Investor Analytics
- Risk Analytics Dashboard

## Key Metrics
- CAGR
- Sharpe Ratio
- Sortino Ratio
- Alpha
- Beta
- Maximum Drawdown
- Fund Score

## Dashboard Pages
1. Executive Overview
2. Fund Performance
3. Investor Analytics
4. Risk Analytics

## How to Run

1. Install dependencies
2. Execute ETL scripts
3. Load data into SQLite
4. Open Power BI dashboard
